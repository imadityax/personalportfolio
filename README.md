## Getting Started

Clone the repository:

```bash
git clone https://github.com/imadityax/personalportfolio.git
```

Move into the project directory:

```bash
cd personalportfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

---
